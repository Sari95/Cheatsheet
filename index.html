Cheatsheet

CSS-Box Model

Eine Box hat in CSS 4 Bereiche, die durch CSS verändert werden können: Content, Border, Padding und Margin.

  • Berechnung der kompletten Breite eines Elements: margin-left + border-left-width + padding-left + width + padding-right + border-right-width + margin-right
  • Berechnung der kompletten Höhe eines Elements: margin-top + border-top-width + padding-top + height + padding-bottom + border-bottom-width + margin-bottom

Content

Hier wird der Inhalt der Box angezeigt. Das heisst, der geschriebene Text und die eingefügten Bilder werden hier angezeigt. Der Content eines Elementes hat 2 Dimensionen:

  • height in Pixel
  • width in Pixel

Padding

Füllt den Raum zwischen dem Content einer Box und dem Rahmen einer Box aus. Kann mit Pixel in padding-top, padding-right, padding-bottom & padding-left einzel definiert werden oder in dieser Reihenfolge in einer Zeile.

Border

Das ist die Linie, welche das Element umrandet. Der Rahmen kann mit width, style & color spezifiziert werden. Für einen perfekten Kreis kann die Höhe & Breite auf die gleiche Pixelgrösse gesetzt werden & der border-radius dabei auf 50% definiert werden.

  • width: Dicke des Rahmens - Kann in Pixel oder mit den Keywords thin, medium, thick definiert werden
  • style: Design des Rahmens - Styles wie none, dotted, solid, hidden, dashed, groove, ridge, inset, outset
  • color: Farbe des Rahmens - In Hexadezimalcode, in RGB oder mit color-Keywords

Margin

Bestimmt die Grösse des Raumes direkt ausserhalb der Box. Kann mit Pixel in margin-top, margin-right, margin-bottom & margin-left einzel definiert werden oder in dieser Reihenfolge in einer Zeile.

Tags & Attribute für Tabellen

Tags

Tag Name Beschreibung
<table> Table Das wrapper Element für die gesamte HTML Tabelle.
<thead> Table Head Das Set von Zeilen, welche die Anzahl an Spalten-headers in einer Tabelle definiert.
<tbody> Table Body Das Set von Zeilen, welche den tatsächlichen Inhalt der Tabelle enthalten.
<tr> Table Row Der Container für die Tabellenzeilen.
<td> Table Data Der Container für die Tabellenzeilen.
<tfoot> Table Foot Das Set von Zeilen, welche den Footer einer Tabelle definieren.

Attribute

Attribut Name Beschreibung
colspan Col Span Definiert, wie viele Spalten ein td Element überspannen soll.
rowspan Row Span Definiert, wie viele Zeilen ein td Element überspannen soll.
